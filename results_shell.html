Overview

This dashboard measures how the prevalence of skills associated with each node in the AI Skill Matrix changes between 2023 and 2026 job postings. The mapping from raw job posting skills to matrix nodes is performed using a semantic similarity approach.

Data Sources

  • Job postings: Lightcast job posting data with pre-extracted standardized skills, filtered by O*NET occupation codes.
  • Skill Matrix: 55 nodes organized into 3 categories (Declining, Enduring, Emerging), each classified as hard or soft skill. The matrix defines a parent-child hierarchy used in the dashboard layout.

Step 1: Building the Skill-to-Node Dictionary

Each of the 55 matrix node names and all 9,201 unique standardized skills (pooled from both 2023 and 2026 data) are encoded into dense vector embeddings using the all-MiniLM-L6-v2 sentence-transformer model.

For each standardized skill, cosine similarity is computed against all 55 node embeddings. The skill is assigned to its highest-scoring node, provided the similarity exceeds a threshold of 0.45. Skills below the threshold are left unmapped.

all-MiniLM-L6-v2Embedding model 0.45Cosine similarity threshold 830Skills mapped (9.0%)

Step 2: Computing Node Penetration

For each combination of (occupation, year, node), we calculate the penetration rate: the percentage of job postings that contain at least one standardized skill mapped to that node.

  1. Each job posting's comma-separated standardized skills are looked up in the dictionary.
  2. A job is counted as "containing" a node if any of its skills map to that node.
  3. Penetration = (number of jobs containing the node / total jobs in that occupation-year group) × 100.

Step 3: Comparing 2023 vs 2026

The change in penetration (in percentage points) is computed for each node per occupation. The dashboard displays both years side by side, with the 2026 bar colored green if penetration increased and red if it decreased.

Limitations

  • Only 9.0% of unique standardized skills exceed the similarity threshold and are mapped to nodes. The remaining skills are not represented in the penetration metric.
  • Each skill maps to exactly one node (the best match). Some skills could reasonably map to multiple nodes.
  • The embedding model captures semantic similarity, not expert-validated skill taxonomies. Some mappings may be imprecise.
  • Job volume grew significantly between 2023 and 2026 across all three occupations, which may reflect market trends or data collection changes rather than skill shifts alone.
